## What's new

- **Nick color picker** — `/color <name|#rrggbb|reset>` sets your nickname color. A
  full HSV color picker modal is also accessible from Settings → Nick Color.

- **VAD Sensitivity screen** — a live spectrogram and RMS level bar replace the
  old preset cycle. Arrow keys adjust the threshold in real time; a ▲ marker
  shows the current trigger point against your voice level.

- **Message line selection** — type `/quote ` or `/pin ` (with a trailing space)
  to enter keyboard cursor mode in the message view. Up/Down navigate lines,
  Enter selects, Escape cancels.
  - `/quote` pre-fills the input bar with the quoted line styled as a blockquote
    (`╷ @nick: …`) so you can type your reply inline.
  - `/pin` pins the selected message to a sticky header at the top of the
    channel, visible to all members.

- **Server-assigned message IDs** — every chat message now carries a `msg_id`
  (UUID4) generated by the server, required for pin targeting.

## Bug fixes

- None.

## Breaking changes

- Server protocol: `S2C.chat` payloads now include `msg_id`. Old clients that
  do not read this field will ignore it silently — no crash.
- New message types `pin_message` (C2S) and `pin_added` (S2C) are unknown to
  old clients and will be dropped silently.
